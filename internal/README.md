# Internal Package Map

- `agent/`: Agent service contracts and provider implementations.
  - Contains the boilerplate service/provider abstraction and a stub provider.
- `tools/`: Primitive tool contracts, registry, and tool implementations.
  - Contains v1 tool names and stub runtime wiring.
- `tui/`: Bubble Tea terminal interface for Luna.
  - Rendering, layout, input handling, and slash command UX.
