# Tools Package

`internal/tools` contains Luna's primitive tool runtime contracts.

## Current status

- Typed tool names for the v1 tool surface.
- Registry for tool lookup and replacement.
- Stub implementations for all default tools.

## Default tools

- `rg_search`
- `read_file`
- `glob`
- `grep`
- `web_search`
- `web_fetch`
- `edit`
- `multi_edit`
